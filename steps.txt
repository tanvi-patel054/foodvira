Step - 2:
Make sure you have installed python
...\(specific folder)\Fudvirapy> python --version

Step - 3 : create virtual environment
...\(specific folder)\Fudvirapy> python -m venv myvenv

Step - 4 : Active and Deactive your virtual env
Active:
...\(specific folder)\Fudvirapy> myvenv\Scripts\activate 

Deactive:
(myvenv) ...\(specific folder)\Fudvirapy> myvenv\Scripts\deactivate 

Step - 5 : Install Django
Before install any module or package make sure your virtual env. is active or not
(myvenv) ...\(specific folder)\Fudvirapy> pip install django

Step - 6 : create requirements.txt for manage modules and package version
(myvenv) ...\(specific folder)\Fudvirapy> type nul > requirements.txt

Step - 7 : add all installed modules and packages inside the requirments.txt
(myvenv) ...\(specific folder)\Fudvirapy> pip freeze > requirements.txt

Step - 8 : Check installed modules and packages inside virtual env.

Step - 9 : create django project
(myvenv) ...\(specific folder)\Fudvirapy> django-admin startproject project .

Step - 10 : run you django project
(myvenv) ...\(specific folder)\Fudvirapy> python manage.py runserver

Step - 11 : create your apps inside "apps" dir
(myvenv) ...\(specific folder)\Fudvirapy> mkdir apps
(myvenv) ...\(specific folder)\Fudvirapy\apps> mkdir accounts
(myvenv) ...\(specific folder)\Fudvirapy\apps> mkdir products
(myvenv) ...\(specific folder)\Fudvirapy\apps> cd ..
(myvenv) ...\(specific folder)\Fudvirapy> python manage.py startapp accounts apps\accounts
(myvenv) ...\(specific folder)\Fudvirapy> python manage.py startapp products apps\products
